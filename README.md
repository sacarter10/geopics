geopics
=======
Search Foursquare venues by location and browse Instagram pictures taken at that location. Try it out at http://geopics.herokuapp.com.

Technologies used:
<ul>
<li> Ruby 2.0.0 </li>
<li> Rails 4 </li>
<li> CoffeeScript </li>
<li> Turbolinks </li>
<li> Slim </li>
<li> EJS templates </li>
<li> Instagram API </li>
<li> Google Maps API </li>
<li> Foursquare API </li>
<li> RestClient </li>
<li> RSpec </li>
<li> Capybara </li>
<li> Twitter Bootstrap 3 </li>
<li> jQuery </li>
<li> Figaro </li>
</ul>
