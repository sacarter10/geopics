geopics
=======
Search Foursquare venues by location and browse Instagram pictures taken at that location. Try it out at http://geopics.herokuapp.com.

Technologies used:
• Ruby 2.0.0
• Rails 4
• CoffeeScript
• Turbolinks
• Slim
• Instagram API
• Google Maps API
• Foursquare API
• RestClient
• RSpec
• Capybara
• Twitter Bootstrap 3
• jQuery
• Figaro
